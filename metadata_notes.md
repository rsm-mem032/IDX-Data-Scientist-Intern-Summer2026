# Metadata Notes

## ClosePrice

Final sold close price for the property transaction. This is the target variable for prediction.

## PropertyType

High-level property category. This project should only include records where `PropertyType` is `Residential`.

## PropertySubType

More specific property classification. This project should only include records where `PropertySubType` is `SingleFamilyResidence`.

## LivingArea

Interior living area of the home, typically measured in square feet.

## BedroomsTotal

Total number of bedrooms reported for the property.

## BathroomsTotalInteger

Total number of bathrooms represented as an integer count.

## LotSizeSquareFeet

Total lot size for the parcel in square feet.

## YearBuilt

Year the residential structure was built.

## City

City where the property is located.

## PostalCode

Postal ZIP code for the property location.

## Latitude

Latitude coordinate for the property location.

## Longitude

Longitude coordinate for the property location.

## CloseDate

Date when the property sale closed.
